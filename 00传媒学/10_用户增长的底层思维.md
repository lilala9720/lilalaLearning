# 传媒学 - 第 10 课：用户增长的底层思维

## 学习目标

学完本课后，你能够：
- 理解增长不是"爆款"，而是一个**可重复的系统**
- 掌握 AARRR 漏斗模型，并能诊断自己的账号卡在哪一层
- 理解"内容-受众匹配"（Content-Audience Fit）为什么比涨粉技巧重要一百倍
- 找到你账号的"啊哈时刻"——观众从"路过"变成"关注"的临界点
- 理解留存 > 拉新，以及为什么大部分创作者把精力花错了地方

## 内容讲解

### 增长不是运气，是系统

肖恩·埃利斯（Sean Ellis）在《增长黑客》（Hacking Growth）里提出了一个让硅谷创业公司奉为圭臬的观点：

> **增长不是一个瞬间事件，而是一个持续优化的系统。**

大部分创作者对"增长"的理解是这样的：

```
拍一条视频 → 祈祷它爆 → 爆了就涨粉 → 没爆就焦虑
```

这不是增长，这是赌博。

真正的增长思维是这样的：

```
理解用户是怎么找到你的 → 他们为什么留下 → 他们为什么会推荐你
→ 在每个环节找到可以优化的杠杆点 → 持续迭代
```

区别在于：**赌博依赖单次爆发，系统依赖可重复的流程。**

一个有增长系统的创作者，即使没有爆款，粉丝也在稳定增长。因为她知道观众从哪来、为什么留下、为什么推荐——每一步都有可以调的旋钮。

### AARRR 漏斗：用户增长的五层结构

埃利斯提出的增长模型叫 AARRR（也叫"海盗指标"，因为读起来像海盗喊叫）。它描述了用户从"第一次接触你"到"帮你传播"的完整路径：

```
┌──────────────────────────────────────────────┐
│  A - Acquisition（获客）                      │
│  观众怎么第一次看到你的视频？                  │
├──────────────────────────────────────────────┤
│  A - Activation（激活）                       │
│  观众看完后，做了什么？（点赞？评论？关注？）  │
├──────────────────────────────────────────────┤
│  R - Retention（留存）                        │
│  关注了之后，她还会回来看你下一条吗？          │
├──────────────────────────────────────────────┤
│  R - Revenue（变现）                          │
│  她愿意为你付费吗？（课程、咨询、打赏）        │
├──────────────────────────────────────────────┤
│  R - Referral（推荐）                         │
│  她会不会把你的视频转给朋友？                  │
└──────────────────────────────────────────────┘
```

**关键洞察：这是一个漏斗。** 每一层都会流失用户。100个人刷到你的视频，可能50个看完，10个点赞，3个关注，1个看你下一条，0.1个付费。

大部分创作者只盯着最顶层（获客）——怎么让更多人看到我？但增长黑客的思维是：**先堵住漏斗下面的洞，再往上面倒水。**

为什么？因为如果你的留存有问题（关注了也不看下一条），那获客再多也是白费——水倒进去全漏了。

### 把 AARRR 翻译成创作者语言

| AARRR 层 | 在创作者场景中 = | 你能优化的东西 |
|----------|------------------|----------------|
| **获客** | 算法把你推给陌生人 | 封面、标题、钩子（前3秒）、发布时间、话题标签 |
| **激活** | 路人变成互动者 | 完播率、点赞率、评论引导、关注引导 |
| **留存** | 粉丝持续观看 | 内容一致性（品牌三角）、更新频率、系列化内容 |
| **变现** | 粉丝愿意付费 | 信任深度、专业度、产品设计 |
| **推荐** | 粉丝帮你传播 | 社交货币、情绪唤醒、实用价值（还记得 STEPPS 吗？） |

看到没有？**你前面学的所有东西都在这张表里**：

- 钩子和前3秒 → 获客层（第04课注意力经济、第08课HCTA）
- 完播率和情绪弧线 → 激活层（第06课剪辑思维、第07课叙事结构）
- 品牌三角和内容一致性 → 留存层（第09课个人品牌）
- STEPPS → 推荐层（第03课传播原则）

增长不是一个独立的技能，它是**把你已有的所有技能串联起来的框架**。

### 内容-受众匹配：比涨粉技巧重要一百倍的东西

在创业领域有个概念叫"产品-市场匹配"（Product-Market Fit）：你的产品是不是一个真实市场真正需要的东西？

在内容创作领域，对应的概念是：

> **内容-受众匹配（Content-Audience Fit）：你的内容，是不是一群真实存在的人正在找的东西？**

很多创作者的问题不是"内容不好"，而是"内容好但没有匹配到对的受众"。或者反过来——她其实不知道自己的受众是谁。

怎么判断你有没有达到内容-受众匹配？埃利斯有一个经典测试：

> **如果明天你不再更新了，会有多少人觉得"很失落"？**

- 如果答案是"几乎没有"——你还没有匹配。内容可能有人看，但没有人"需要"你。
- 如果答案是"有一小群人会很失落"——你已经匹配了，只是规模还小。这时候才值得发力增长。

**在匹配之前追求增长，就像在产品还没做好之前就投广告——越多人来，差评越多，死得越快。**

拿你的情况来说：
- 你的 NPD 视频数据好 → 说明"关系中的心理认知转换"这个方向有受众
- 你的评论区有人说"终于有人把这个说清楚了" → 这是匹配的信号
- 你的品牌三角已经明确了（翻译官+观察者 × 认知转换解决关系问题 × 锐评口语风格） → 这是匹配的基础

你现在可能处于匹配的早期阶段。接下来要做的不是"怎么涨粉"，而是**验证和强化这个匹配**——持续用品牌三角产出内容，观察数据反馈，看匹配是否稳定。

### 啊哈时刻：观众从"路过"变成"关注"的临界点

增长黑客里有一个核心概念叫"啊哈时刻"（Aha Moment）——用户第一次感受到产品核心价值的瞬间。

- Facebook 的啊哈时刻：新用户在10天内加了7个好友 → 留存率暴增
- Dropbox 的啊哈时刻：用户第一次在另一台设备上看到自己的文件自动同步
- Twitter 的啊哈时刻：用户关注了30个人之后，信息流变得有趣 → 开始留下来

对于内容创作者来说：

> **你的啊哈时刻 = 新观众第一次觉得"她说的就是我的情况"的那个瞬间。**

这个瞬间一旦发生，她就从"随便刷到"变成"我要关注她，怕以后找不到"。

你的啊哈时刻可能是什么？猜一下：
- 不是你讲了一个心理学概念（这个很多人能做）
- 而是你用**锐评的方式说出了一个她心里知道但没人说出来的关系真相**

"孝顺是两个词"这个选题为什么好？因为它有极强的啊哈时刻潜力——很多人一辈子没想过孝和顺是两回事。你说出来的那一秒，她的认知就被打开了。

**怎么找到你的啊哈时刻？**

不用猜，看数据：
1. 找你关注转化率最高的视频（不是播放量最高的，是"看完后关注比例"最高的）
2. 分析那条视频里，观众在评论区说的最多的话是什么
3. 那句话背后的感受，就是你的啊哈时刻

### 留存 > 拉新：大部分创作者花错精力的地方

这是本课最违反直觉的观点：

> **涨粉最有效的方式，不是拉新粉，而是留住老粉。**

为什么？三个原因：

**原因一：留存驱动推荐。**

一个持续看你视频的粉丝，会在朋友遇到关系问题时说"你去看XXX的视频"。这种口碑推荐的转化率远高于算法推荐。而且它是免费的。

留存 → 信任 → 推荐 → 新粉丝 → 她们也留存 → 她们也推荐 → ……

这是增长飞轮，不是增长漏斗。飞轮一旦转起来，越转越快。

**原因二：留存影响算法。**

短视频平台的算法核心逻辑很简单：**你的内容能不能留住用户在平台上？** 如果你的粉丝每次都看完你的视频，算法就会认为你的内容有价值，给你更多推荐流量。

粉丝看完率高 → 算法推更多 → 更多新人看到 → 其中一部分关注 → 她们也看完 → 算法推更多 → ……

又一个飞轮。

**原因三：留存筛选真正的受众。**

不是所有关注你的人都是你的"受众"。有些人关注你是因为某条爆款视频戳中了她一次，但她其实对你的核心议题没有持续兴趣。

留存率会自动帮你筛掉这些人，留下真正跟你品牌匹配的核心受众。这群人虽然数量少，但价值极高——她们会互动、会转发、会付费。

**这就是为什么你在第09课建立品牌三角如此重要。** 品牌一致性是留存的前提——观众留下来是因为她相信"下一条视频也会给我类似的价值"。如果你今天讲关系，明天讲美食，粉丝的预期被打破，留存就崩了。

### 增长杠杆：在你的漏斗里找到最值得优化的那一层

增长黑客的核心方法论不是"什么都优化"，而是**找到当前卡点，集中精力突破**。

怎么找卡点？看漏斗每一层的数据：

```
获客：播放量
  ↓ （如果这里断崖 → 钩子/封面/标题有问题）
激活：完播率、互动率
  ↓ （如果这里断崖 → 内容结构/情绪弧线有问题）
留存：粉丝视频的平均观看量 / 粉丝取关率
  ↓ （如果这里断崖 → 品牌一致性/更新频率有问题）
推荐：转发率、收藏率
  ↓ （如果这里断崖 → 实用价值/社交货币不足）
```

**举个例子：**

假设你有一条视频播放量 10 万，但关注转化只有 0.1%。问题出在哪？

不是获客——10万播放说明算法推了。问题在激活或留存层。可能是：
- 视频内容好，但**没有明确引导关注**
- 视频太好笑/太猎奇，吸引了大量非目标受众，她们看完不会关注
- 你的主页其他视频跟这条风格不一致，她们点进主页一看"不是我的菜"就走了

诊断出卡点之后，优化就有方向了。这比盲目追求下一个爆款有效得多。

### 增长实验思维：小步测试，数据说话

增长黑客还有一个核心理念：**不要假设，要测试。**

你觉得"钩子用反问句更好"？别猜，发两条视频，一条用反问，一条用陈述，看数据。
你觉得"发布时间晚上8点更好"？别猜，试一周8点，试一周10点，看数据。

这叫 A/B 测试思维。它的本质是：**用最小的成本验证一个假设。**

对创作者来说，每条视频就是一次实验。但大部分创作者从来不回顾实验结果——发完就发完了，从来不分析"这条为什么好/差"。

建立一个简单的记录习惯就行：

```
视频：[标题]
发布时间：
3天后数据：播放 / 完播率 / 点赞 / 评论 / 关注转化
假设验证：这条我测试的是什么？结果如何？
下一步：基于这条的数据，下一条我要调整什么？
```

你不需要做复杂的数据分析。只需要**有意识地记录和回顾**，就已经超过 90% 的创作者了。

## 小结

1. **增长是系统，不是爆款**——可重复的流程比一次性的运气有用一百倍
2. **AARRR 漏斗**：获客→激活→留存→变现→推荐，每一层都有可优化的杠杆
3. **内容-受众匹配是前提**——在匹配之前追求增长，等于往破桶里倒水
4. **找到你的啊哈时刻**——观众从路过变成关注的临界瞬间，藏在你转化率最高的视频里
5. **留存 > 拉新**——留住老粉驱动推荐飞轮、影响算法推荐、筛选核心受众
6. **找卡点，不要什么都优化**——看漏斗每层的数据，集中突破断崖处
7. **每条视频都是一次增长实验**——记录、回顾、迭代
