# 传媒学 — 学习路线

> 基于清华新闻学融媒体传播方向培养方案定制，面向心理学短视频内容创作者。

## 整体架构：5大模块 + 实战训练

```
模块一：传播学底层认知        ← 地基，理解"内容为什么被传播"
模块二：影像叙事与视听语言    ← 技术面，提升视频"质感"和"节奏"
模块三：社会化媒体运营        ← 运营面，把好内容变成影响力
模块四：媒介素养与批判思维    ← 护栏，守住"赋能而非操控"的底线
模块五：数据思维与分析        ← 辅助面，用数据驱动选题和优化
```
