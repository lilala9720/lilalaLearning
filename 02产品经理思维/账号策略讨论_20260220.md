# 账号策略讨论 - 2026.02.20

## 一、用户画像（已确认）

> 32岁左右已婚职场女性。白天上班，晚上带娃，老公指望不上。晚上10点孩子睡了，躺在床上刷手机，这是她一天里唯一属于自己的10分钟。她的核心问题是分不清"这到底是不是我的错"——在婚姻里、在原生家庭里、在职场里，都是同一个模式。她需要的是被看见、被确认"你没有错"。但她自己觉得她需要的是老公听话、孩子乖、钱更多。

**一个内核，三个场景：** 婚姻 / 原生家庭 / 职场，轮着用，底层是同一件事。

## 二、内容定位的边界

**我的角色：举镜子，不替她活。**

- 第一层（免费内容）：被看见 → "终于有人懂我" → 流量
- 第二层（免费内容）：被命名 → "原来这个东西叫这个名字" → 信任
- 第三层（付费产品）：被陪走 → "我想试试" → 收入

不推任何人改变。她自己决定要走了，我提供拐杖。

## 三、选题公式

**场景 + 她说不出口的那句话**

| 场景 | 她不敢说的话 | 选题方向 |
|------|------------|---------|
| 婚姻 | "我累了但不敢停" | 你不是不够好，你是太累了 |
| 原生家庭 | "我恨我妈但我觉得不该恨" | 恨父母不是不孝，是你在长大 |
| 职场 | "我不想扛了但怕被说不负责" | 你扛的不是责任，是别人甩给你的锅 |

### 选题思考流程（完整版）

```
她心里的话 → 为什么不敢说？ → 标题（她的视角）
                            → 或者：对方在干嘛/为什么？ → 标题（对立视角）
```

**关键发现：对立视角 + 揭示动机 = 爆款公式。**

数据验证：NPD为什么要打压你 = 2000+赞（其他视频最多80赞，断层式领先）。

本质：回答了用户最深的问题——"这不是我的错"，而且用最有力的方式——拆穿对方的动机。

### 注意事项

- 找到"她说不出口的话"后，不要跳去升华（给建议/鸡汤/描述），要追问"为什么不敢说"
- 标题标准：她看到会"咯噔"，不是"有道理"
- 两种视角交替使用，不要只做一种

## 四、关于客单价和人群选择

**结论：不换人群，在现有人群里找能付钱的那一层。**

- "分不清边界"≠ 穷。很多职场女性收入不低，只是钱花在了家庭上
- 她们缺的不是钱，是"允许自己为自己花钱"的许可
- 付费产品定位对了，客单价几千块可以做到

## 五、两条业务线（不冲突）

| | 内容账号（To C） | 企业培训（To B） |
|--|--|--|
| 对象 | 分不清边界的职场女性 | 律所、公司 |
| 产品 | 免费视频 → 付费课/咨询 | 线下/线上培训 |
| 获客方式 | 短视频推流 | 转介绍、直接谈、见面礼 |
| 卖点 | "她懂我" | "心理学+销售，市面上没有" |

**内容是品牌，培训是生意。品牌吸引C端，偶尔引来B端。不为偶尔来的B端改品牌方向。**

## 六、当前数据现状

- 三个平台共约800粉丝
- 25条视频试错数据

- 核心问题：标题和开头没圈人，推流不精准（包装问题，非方向问题）
- **断层爆款：NPD为什么要打压你（2000+赞，其他最多80赞）**

## 七、三种内容武器（痛点/痒点/爽点）

| 武器 | 给她什么 | 她的反应 | 作用 |
|------|---------|---------|------|
| 痛点：你没错 | 释放 | "终于有人说了"→ 转发 | 拉新、爆款 |
| 痒点：你可以这样 | 心动 | "我也想试试"→ 关注 | 涨粉、粘性 |
| 爽点：你试试这个 | 方法 | "我今天就能用"→ 收藏 | 信任、转付费 |

**痛点引流、痒点种草、爽点转化。三种交替用。**

## 八、商业模式内核（核心顿悟）

> **我不生产道理，我把道理接上她的生活。**

用户不缺知识——她刷过一万条短视频，"边界感""讨好型人格""原生家庭"这些词她都见过。她缺的是有人帮她把这些散落的碎片接到她正在经历的具体困境上。

- 免费内容 = 命名："原来你的问题叫这个名字"
- 付费产品 = 连接："知道名字之后，你具体怎么走出来"

**知识不值钱，连接才值钱。** 这是咨询、心理咨询、教育行业共通的底层逻辑。

与所有"教心理学的博主"的区别：她们在生产道理，我在把道理接上用户的生活。

痒点的关键：不画大饼（"爱自己就会遇到对的人"），画小画面（"第一次挂掉你妈的电话，你会慌，但十分钟后你会笑"）。
