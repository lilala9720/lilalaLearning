# 产品经理思维 - 学习进度

## 学习目标
用产品经理的方法论来经营个人账号和服务，把"自己"当产品来打造。

## 学习路线（共 10 课）

| 课序 | 主题 | 状态 | 掌握度 | 备注 |
|------|------|------|--------|------|
| 01 | 产品思维是什么：从"我有什么"到"他要什么" | 待学习 | - | 思维转换的起点 |
| 02 | 用户研究：你到底在服务谁 | 待学习 | - | 用户画像与细分 |
| 03 | 需求分析：痛点、痒点、爽点 | 待学习 | - | 梁宁需求三层次 |
| 04 | 价值主张：凭什么选你不选别人 | 待学习 | - | 差异化定位 |
| 05 | MVP 思维：先做出来，再做好 | 待学习 | - | 最小可行产品/服务 |
| 06 | 用户体验：服务流程就是产品设计 | 待学习 | - | 触点管理与体验地图 |
| 07 | 数据思维：别猜，去量 | 待学习 | - | 关键指标与反馈闭环 |
| 08 | 迭代优化：小步快跑，持续进化 | 待学习 | - | PDCA 与敏捷思维 |
| 09 | 增长策略：从 0 到 1 再到 N | 待学习 | - | 获客、留存、转化 |
| 10 | 商业模式：把价值变成收入 | 待学习 | - | 个人商业画布 |
