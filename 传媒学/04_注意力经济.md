# 传媒学 - 第 4 课：注意力经济

## 学习目标

学完本课后，你能够：
- 理解"注意力"为什么是这个时代最稀缺的资源，以及它是如何被"交易"的
- 说出注意力经济的基本商业逻辑：谁在买注意力？谁在卖？你在这个链条里的位置是什么？
- 理解短视频平台的算法本质上是一个"注意力分配系统"
- 重新审视"完播率""停留时长"等指标背后的经济逻辑
- 思考作为创作者，你是在"赢得"注意力还是在"收割"注意力

## 内容讲解

### 一个你每天都在参与的交易

你每天打开抖音或小红书，刷30分钟视频。这30分钟里，你没有花一分钱。

但你付了钱。你付的是**注意力**。

平台把你的注意力打包卖给了广告商。广告商付真金白银买你的眼球停留。这就是整个短视频行业的商业模型——**用户用注意力付费，平台用注意力变现。**

吴修铭（Tim Wu）在《注意力商人》（The Attention Merchants）里梳理了一个跨越百年的历史：从19世纪的廉价报纸，到广播、电视，再到互联网和短视频——**每一次媒介革命，本质上都是一次注意力收割技术的升级。**

而你，作为一个短视频创作者，处在这个链条的一个非常微妙的位置上。

### 注意力经济的三角关系

```
        平台（抖音/小红书）
       /        \
      /          \
   分配流量      卖广告位
    /              \
创作者 ——————————→ 用户
  生产内容     消费注意力
```

三个角色各自的利益：

| 角色 | 它想要什么 | 它怎么得到 |
|------|-----------|-----------|
| **平台** | 用户的总停留时长（这是它卖给广告商的"库存"） | 用算法把最"黏人"的内容推给最可能停留的用户 |
| **广告商** | 用户的注意力和购买行为 | 向平台付费，插入广告 |
| **创作者（你）** | 流量、粉丝、变现 | 生产能留住用户注意力的内容 |
| **用户** | 娱乐、信息、情绪价值 | 用注意力"付费"，换取免费内容 |

注意一个关键事实：**平台的核心指标是"用户总停留时长"，不是"用户学到了多少"或"用户变得更好了没有"。**

这意味着什么？意味着算法天然偏好能**最大化停留时长**的内容，而不是最有价值的内容。这两者有时候重合，但很多时候不重合。

### 算法：注意力的分配系统

你可能觉得算法很神秘。其实它的底层逻辑非常简单：

> **把对的内容，在对的时间，推给对的人，让他/她停留尽可能久。**

算法用来判断"对不对"的核心指标：

| 指标 | 算法怎么理解它 | 对你的含义 |
|------|---------------|-----------|
| **完播率** | 这个内容能不能留住人看完 | 你的内容必须在前3秒抓住人，中间不能让人走神 |
| **互动率**（点赞/评论/转发） | 这个内容能不能引发行为 | 你的内容必须激发"想说点什么"的冲动 |
| **停留时长** | 用户在这个内容上花了多少时间（包括看评论） | 评论区本身也是内容的一部分 |
| **完播后行为** | 看完之后用户是继续刷还是退出APP | 如果你的内容让人看完就退出，算法会惩罚你 |
| **负反馈**（不感兴趣/举报） | 这个内容是否让用户不舒服 | 过度标题党会被负反馈反噬 |

看到了吗？**这些指标衡量的全是"注意力的黏性"，不是"内容的质量"。** 一条让人愤怒到忍不住骂人的视频，在算法眼里和一条让人感动到流泪的视频，可能有同样高的互动率。算法不区分情绪的性质，只看数字。

### 回到你的两条视频

现在用注意力经济的框架重新看你分析过的两条视频：

**火锅视频（数据好）：**
- 前3秒：场景代入（你跟男朋友吃火锅），完播率有保障
- 中间：情绪持续升级（表白→被弹开→愤怒），注意力不会断
- 结尾：没有给出完整解决方案，观众带着情绪去评论区——**停留时长被评论区延长了**
- 算法判断：完播率高 + 互动率高 + 停留时长长 = 继续推

**你的育儿视频（数据差）：**
- 前3秒："育儿观不同本质是原生家庭不同"——这是一个**概念**，不是一个**场景**。部分观众还没建立代入感就划走了
- 中间：情绪先升后降，你把问题分析清楚还给了解决方案——观众觉得"嗯，有道理"
- 结尾：问题解决了，没有悬念，没有驱动力去评论区讨论
- 算法判断：完播率一般 + 互动率低 + 停留时长短 = 停止推送

**这不是你内容"不好"，是你的内容不符合注意力经济的分配逻辑。** 这是两回事。

### 注意力的三个层次

不是所有注意力都是平等的。可以分为三层：

| 层次 | 特征 | 对应你的场景 |
|------|------|-------------|
| **捕获注意力** | 让人停下来看你（0-3秒） | 封面、标题、开头第一句话 |
| **保持注意力** | 让人看完（3秒到结尾） | 叙事节奏、情绪曲线、信息密度 |
| **转化注意力** | 让人看完后做点什么（关注/评论/转发/购买） | 行动号召、开放式结尾、情绪余味 |

大多数创作者只想着第一层（怎么让人点进来），稍好一点的想到第二层（怎么让人看完），但真正拉开差距的是第三层（看完之后发生什么）。

你的育儿视频在第三层出了问题——看完之后，观众没有理由去做任何事。问题已经被你解决了，情绪已经被你安抚了，评论区自然冷清。

而火锅视频在第三层非常强——看完之后，观众带着愤怒冲进评论区骂人、讲自己的故事、互相安慰。**第三层的注意力是免费的额外收益，但它往往决定了一条视频的生死。**

### "赢得"注意力 vs "收割"注意力

吴修铭在书里反复强调一个历史规律：

> **每一代注意力商人最终都会走向过度收割，然后引发用户的反抗。**

廉价报纸靠耸动标题吸引眼球，最终失去公信力。电视靠垃圾节目填充时段，最终被有线电视和互联网颠覆。每次都是同样的剧本：**短期收割注意力的手段，长期会损耗信任。**

这对你来说意味着什么？

作为心理学内容创作者，你面临一个持续的选择：

| | 收割注意力 | 赢得注意力 |
|---|---|---|
| **做法** | 标题党、制造焦虑、过度简化、情绪操控 | 真实故事、有用方法、坦诚局限性 |
| **短期效果** | 数据可能更好 | 数据可能一般 |
| **长期效果** | 粉丝粘性低，信任耗损，需要不断加大刺激 | 粉丝粘性高，信任积累，口碑增长 |
| **核心区别** | 用户看完觉得"被耍了" | 用户看完觉得"被帮到了" |

你在第03课说过"只要没骗人就行，只是换了个壳子"——用注意力经济的框架来看，"换壳子"本身不是问题，问题是：你换的那个壳子是在**赢得**注意力还是在**收割**注意力？如果壳子是一个好故事，让人愿意听完你的观点，那是赢得；如果壳子是一个恐吓性的标题，让人点进来发现和预期不符，那是收割。

### 注意力的有限性：你在跟谁抢？

最后一个重要的认知：注意力是零和博弈。

用户每天的注意力总量是有限的。你拿走了30秒，别人就少了30秒。你的竞争对手不只是同领域的心理学创作者，而是**所有争夺同一个用户同一个时间段注意力的内容**——搞笑视频、美食探店、明星八卦、游戏直播……

这意味着：
- 你的内容不是在跟"其他心理学视频"比好不好，而是在跟"用户此刻可以看的一切内容"比**谁更值得花这30秒**
- 用户划走你的视频不是因为你"不好"，而是因为旁边有一条更"值得"的内容
- "值得"不是你定义的，是用户的大脑在0.5秒内根据直觉判断的

这也解释了为什么"概念先行"的开头在注意力经济中天然吃亏——大脑在0.5秒内无法判断一个抽象概念"值不值得花时间"，但可以瞬间判断一个具体场景"跟不跟我有关"。

## 小结

1. **注意力是这个时代最稀缺的资源**，短视频平台的本质是"注意力的中间商"——用算法分配注意力，向广告商出售注意力
2. **算法的核心指标是停留时长和互动率**，它衡量的是注意力的黏性，不是内容的质量。理解这一点才能理解为什么有些好内容数据差
3. **注意力有三个层次**：捕获（前3秒）→ 保持（完播）→ 转化（看完后的行为）。第三层往往决定一条视频的生死
4. **"赢得"注意力和"收割"注意力是两种路径**，短期收割可能有效，但长期会耗损信任。你的壳子是好故事还是恐吓标题，决定了你走的是哪条路
5. **注意力是零和博弈**，你的竞争对手不只是同行，而是用户此刻能看的一切内容
