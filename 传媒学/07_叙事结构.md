# 传媒学 - 第 7 课：叙事结构

## 学习目标

学完本课后，你能够：
- 理解为什么人类大脑天然偏好"故事"而非"道理"
- 掌握三幕结构的基本原理，并能在60秒短视频中应用
- 理解"冲突"是一切叙事的发动机——没有冲突就没有故事
- 学会区分"在讲故事"和"在讲道理配了个例子"
- 用叙事结构重新设计你已有的内容

## 内容讲解

### 为什么你讲道理没人听，讲故事就有人听？

你一定遇到过这种情况：你精心总结了一个心理学知识点，逻辑清晰、层次分明，发出去没什么人看。但另一个创作者讲了一个跟闺蜜吵架的故事，顺带提了一句同样的知识点，数据好几倍。

为什么？

因为**人类的大脑不是为"接收信息"设计的，是为"理解故事"设计的。**

这不是比喻。认知神经科学的研究发现：
- 当你听到一组数据或一个抽象概念时，大脑只有**语言处理区**在工作
- 当你听到一个故事时，大脑的**运动皮层、感觉皮层、情绪中枢**全部被激活——就好像你亲身经历了那件事一样

故事让大脑"全开"，道理只让大脑"局部开"。这就是为什么同样的信息，故事比道理有效得多。

罗伯特·麦基（Robert McKee）在《故事》里说：

> **故事是人类对生命体验进行理解和排序的基本方式。它不是娱乐，它是认知工具。**

### 一切故事的发动机：冲突

麦基认为，故事的本质不是"一系列事件"，而是**"围绕冲突的一系列事件"**。

没有冲突 = 没有故事。

什么是冲突？**角色想要某样东西，但有障碍挡在面前。** 就这么简单。

| 冲突类型 | 含义 | 你的内容中的例子 |
|---------|------|----------------|
| **人vs人** | 和另一个人对抗 | 你vs你爸（NPD视频）、你vs你老公（育儿视频） |
| **人vs自己** | 内心矛盾、自我怀疑 | "我一开始以为是育儿观念不同，后来发现是我在找补" |
| **人vs环境** | 社会规则、文化压力 | "所有人都说要科学育儿，但我老公家是放养的" |

你的NPD视频为什么有故事感？因为它有清晰的冲突：

> **一个女儿想要被爸爸认可（欲望），但爸爸只会打压她（障碍）。**

这一句话就够了。观众立刻想知道：后来怎么样了？她怎么处理的？冲突解决了吗？

你的育儿视频为什么故事感弱？因为冲突是**模糊的**：

> "我和老公因为育儿观念吵架"——谁想要什么？障碍是什么？不够清晰。

如果改成：

> "我只有一个要求：我说的话，你要当回事。但他从来不当回事。"——冲突立刻锐利了。

**冲突越具体、越锐利，故事的牵引力越强。**

### 三幕结构：最古老也最有效的叙事框架

从亚里士多德到好莱坞，人类讲故事的基本结构从来没变过：

```
第一幕：建置（Setup）——这是谁？发生了什么事？
     ↓ 【激励事件：打破日常的那一下】
第二幕：对抗（Confrontation）——面对冲突，挣扎、尝试、失败
     ↓ 【高潮：最关键的转折】
第三幕：解决（Resolution）——冲突的结果，主角的变化
```

三幕结构的核心不是"开头中间结尾"——任何东西都有开头中间结尾。它的核心是：

1. **第一幕建立"日常"，然后用激励事件打破它** — 观众需要先知道"正常状态"是什么，才能感受到"被打破"的冲击
2. **第二幕是最长的部分，充满挣扎和障碍** — 如果主角轻松解决了问题，就没有故事了。挣扎和障碍是让观众留下来的原因
3. **第三幕给出结果，但不一定是"好结果"** — 重要的是主角有了变化，哪怕是认知上的变化

### 60秒里怎么用三幕结构？

你可能觉得：三幕结构是电影用的，60秒短视频哪有空间？

其实60秒完全够。麦基说过：**故事的长度不取决于时间，取决于冲突的复杂度。** 一个简单的冲突，60秒就能讲出完整的三幕。

用你的NPD视频做拆解：

| 幕 | 时间占比 | 内容 | 功能 |
|---|---------|------|------|
| **第一幕** | ~15% | "我爸是NPD。我哭着问他：你为什么只说我的缺点？他说：我这是为你好。" | 建置日常+激励事件（哭着问爸爸=打破日常的那一下） |
| **第二幕** | ~60% | "他从来不改。直到我学了NPD才知道——打压是他的刚需，他的自我存在感极低……" | 对抗（不是行为上的对抗，而是认知上的挣扎：从"他为什么这样"到"我终于理解了"） |
| **第三幕** | ~25% | "解释、反击对NPD根本没用……真正自信的人不需要踩任何人。" | 解决（不是关系修复了，而是主角——你——的认知发生了变化） |

注意：**第三幕的"解决"不是"问题解决了"，而是"主角变了"。** 你爸还是NPD，但你不再被困在"为什么他不认可我"的循环里了。这就是故事的力量——观众要的不是happy ending，是**角色弧光**。

### "讲故事"vs"讲道理配了个例子"

这是你最需要区分的一件事。

| | 讲故事 | 讲道理配例子 |
|---|---|---|
| **结构** | 冲突 → 挣扎 → 变化 | 观点 → 举例 → 总结 |
| **主角** | 有一个具体的人在经历这件事 | 没有主角，例子是工具 |
| **情绪弧线** | 有起伏（代入 → 紧张 → 释放） | 平的（"比如说……""举个例子……"） |
| **观众的心态** | "然后呢？后来怎么样了？" | "嗯，有道理"（划走） |
| **你的NPD视频** | ✅ 讲故事 | |
| **你的育儿视频** | | ✅ 讲道理配了例子 |

你的育儿视频虽然有你和老公的具体事例（遛娃、玩手机），但这些事例是**服务于"育儿观不同本质是原生家庭不同"这个观点**的。观众从第一秒就知道你要讲什么道理，剩下的只是论证过程。没有悬念，没有"然后呢？"的牵引力。

如果改成故事结构：

> **第一幕**："上周六晚上，我老公又在宝宝面前玩手机。我说了他一句，他说'偶尔看一次没关系'。然后我们大吵了一架。"（具体场景+冲突爆发）
>
> **第二幕**："我当时气得不行，觉得他根本不在乎我说的话。但是冷静下来以后我问了自己一个问题——如果专家说偶尔看一次确实没关系，我会不会不生气？答案是我还是会生气。"（挣扎+自我追问）
>
> **第三幕**："我突然意识到，我生气的根本不是手机。是从小到大，我爸说话的时候我只能听着，我的意见从来没有被当回事。我老公不听我的时候，触发的是这个旧伤。"（认知变化=角色弧光）

同样的信息，同样的结论，但从"道理配例子"变成了"一个人在经历中发现了真相"。**观众不是在被教育，而是在跟你一起经历这个发现过程。**

### 角色弧光（Character Arc）：观众真正买单的东西

麦基说：**故事的终极吸引力不是情节，是角色的变化。**

角色弧光 = 主角在故事开始时是一种人，在故事结束时变成了另一种人（或至少有了新的认知）。

你的NPD视频的角色弧光：
- 开始：一个不理解爸爸为什么打压自己、被困在痛苦里的女儿
- 结束：一个理解了NPD机制、不再自我怀疑、能说出"真正自信的人不需要踩任何人"的女性

**这个变化过程就是观众最想看到的东西。** 因为你的变化给了她们希望——"如果她能从那样的原生家庭里想通，我是不是也可以？"

你做心理学内容最大的优势就在这里：**你自己的成长经历就是角色弧光。** 从不懂到懂、从痛苦到想通、从被困住到走出来——每一段经历都是一个天然的故事结构。

但你目前的大部分视频没有利用这个优势，因为你习惯性地跳过了"挣扎"的部分，直接给出"想通后的结论"。观众只看到了弧光的终点，没有看到弧线本身。**没有经历过程的结论，就只是一句道理。**

### 叙事结构与前面课程的关系

到这里，模块二的三课（视听语言、剪辑节奏、叙事结构）可以串起来了：

```
叙事结构（第07课）= 你讲什么
     ↕ 相互服务
剪辑节奏（第06课）= 你怎么控制时间
     ↕ 相互服务
视听语言（第05课）= 你怎么控制画面和声音
```

三者的关系：**叙事结构决定了情绪曲线的走向，剪辑节奏控制情绪曲线的速度，视听语言强化情绪曲线的强度。** 它们是同一条情绪曲线的三个维度。

## 小结

1. **人类大脑为故事而生**——故事激活全脑，道理只激活语言区。同样的信息，故事比道理有效得多
2. **冲突是故事的发动机**——角色想要什么+什么挡在面前。冲突越具体越锐利，牵引力越强
3. **三幕结构**：建置（打破日常）→ 对抗（挣扎和障碍）→ 解决（角色变化）。60秒完全够用
4. **"讲故事"≠"讲道理配例子"**——区别在于有没有冲突驱动、有没有主角在挣扎、观众有没有在问"然后呢"
5. **角色弧光是观众真正买单的东西**——不要跳过挣扎直接给结论，过程比结论重要
6. **叙事结构、剪辑节奏、视听语言是情绪曲线的三个维度**，互相服务
