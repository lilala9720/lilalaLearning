# 传媒学 - 第 9 课：个人品牌建设——从创作者到品牌

## 学习目标

学完本课后，你能够：
- 理解"个人品牌"的本质——它不是包装，是一个**承诺**
- 找到自己的品牌三角（身份定位 × 核心议题 × 表达风格）
- 理解"差异化"不是与众不同，而是**可被辨认**
- 学会用"品牌过滤器"做选题决策

## 内容讲解

### 品牌不是你说你是谁，是别人记住你什么

奥斯汀·克莱恩（Austin Kleon）在《一个人的媒体帝国》（Show Your Work）里说了一句很朴素的话：

> **你不需要成为天才。你只需要把你的过程分享出来。**

这句话对你特别重要，原因后面会讲。先说品牌这个词到底是什么意思。

很多人以为品牌 = 人设 = 包装。不是的。

**品牌的本质是一个承诺。** 观众点开你的视频之前，心里已经有了一个预期——"看她的视频我会获得什么"。这个预期就是你的品牌。

举个例子：
- 你打开李永乐老师的视频，预期是"用简单的话讲清楚一个复杂的科学问题"
- 你打开papi酱的视频，预期是"吐槽日常，笑完觉得说的有道理"
- 你打开一个NPD科普号的视频，预期是"帮我理解我爸/我妈/我前任为什么那样"

**品牌 = 观众对你的稳定预期。**

如果你每条视频给观众的感觉都不一样——今天讲NPD，明天讲美食，后天跳舞——观众没办法形成预期，也就没办法"关注"你。关注的本质是"我相信你以后还会给我类似的东西"。

### 你的品牌不需要"造"，需要"发现"

这是新手最大的误区：觉得要先"设计"一个人设，然后去"扮演"。

克莱恩的观点正好相反：**你已经是某个领域的独特存在了，你需要做的不是发明，是发现和放大。**

怎么发现？问自己三个问题：

#### 问题一：你反复在讲什么？

回看你所有的视频（包括没发的草稿），哪些话题你反复提到？哪些主题你不需要查资料就能讲得很投入？

这不是你"应该"做什么内容，而是你**已经在**做什么。你的选题历史就是你的品牌雏形。

你的情况：NPD、原生家庭创伤、亲密关系中的心理模式、能量层级——这些都指向一个方向：**关系中的心理真相**。

#### 问题二：别人因为什么来找你？

不是你觉得自己擅长什么，而是别人认为你擅长什么。看你的评论区、私信、朋友找你聊的话题。

如果你的粉丝经常在评论区说"终于有人把这个说清楚了""你说的就是我妈"——说明你的品牌核心是**把复杂的心理现象翻译成普通人听得懂的话**。

#### 问题三：你愿意一直做下去的是什么？

品牌需要时间建立。如果一个方向你做三个月就腻了，那不是品牌，是尝鲜。

真正的品牌方向应该是你**不挣钱也会继续聊**的东西。因为持续产出的前提是热情，而不是纪律。

### 品牌三角：让"你是谁"变得具体

发现了方向之后，需要把它具体化成一个可操作的框架。我叫它"品牌三角"：

```
        身份定位
       /        \
      /          \
核心议题 ——— 表达风格
```

#### 1. 身份定位：你是以什么身份在说话？

不是"心理学博主"——这太宽了，每个心理学号都能用。身份定位需要回答：**你从哪个角度看心理学？**

几种常见的身份角度：

| 身份角度 | 说话方式 | 典型表达 |
|----------|----------|----------|
| 专家/学者 | 自上而下教学 | "心理学研究表明……" |
| 亲历者 | 平视，自我暴露 | "我爸是NPD，我花了十年才走出来" |
| 翻译官 | 把术语变人话 | "说白了就是……" |
| 观察者/评论者 | 旁观者视角，点评现象 | "你有没有发现，XXX其实是……" |

你之前说"想少提自己，多评论别人"。这说明你正在从"亲历者"往"观察者/评论者"方向移动。但你最火的内容（NPD视频）恰恰是亲历者身份最重的那条。

这不矛盾。**你可以组合身份：以观察者的视角讲故事，偶尔用亲历者的身份放大情绪。** 但主打身份需要明确——观众需要知道"她是那个XXX的人"。

#### 2. 核心议题：你反复在解决什么问题？

核心议题不是"心理学"——太大了。也不是"NPD"——太窄了。

好的核心议题是一个**中间层**：比大方向窄、比具体选题宽，能覆盖你大部分内容。

```
太宽：心理学
刚好：关系中的心理真相 / 被忽略的情感模式 / 原生家庭如何塑造你
太窄：NPD父亲
```

核心议题的作用：**它是你的选题过滤器。** 一个选题再好，如果不在你的核心议题范围内，就不该做。因为做了会稀释品牌。

举个例子：
- "能量层级决定反应模式" → 跟"关系中的心理真相"有关吗？有。能量低的时候在关系中的反应会不同。 → 可以做
- "如何高效背单词" → 跟你的核心议题有关吗？没有。→ 不做，哪怕你真的有方法

### 表达风格：你说话的方式是品牌的一部分

内容可以被模仿，风格很难。

表达风格包括：

**语言层面：**
- 你说话是偏理性分析还是偏感性叙事？
- 你用不用"你们""姐妹们"这种称呼？
- 你喜欢反问还是陈述？
- 你的标志性表达是什么？（比如某个口头禅、某种句式）

**视觉层面：**
- 你的镜头风格（近景为主？有没有标志性的布景？）
- 你的文字排版风格
- 你的配色倾向

**情绪基调：**
- 有的人永远温柔（温暖型）
- 有的人永远犀利（锐评型）
- 有的人在温柔和犀利之间切换（反差型）

你目前的风格线索：直接、不绕弯、敢说、带点锐度。这其实已经是一种辨识度了。

### 差异化：不是奇怪，是可被辨认

很多创作者一提"差异化"就想着要多特别、多与众不同。但差异化不是奇装异服。

**差异化 = 当观众看到这条视频，不看头像就知道是你。**

怎么做到？不是靠一个绝招，而是靠**品牌三角的组合**：

- 单独看"心理学内容"——没差异化，几百个号都在做
- 单独看"犀利风格"——也没差异化，很多人都犀利
- 但"用犀利的观察者视角，讲关系中被忽视的心理模式，偶尔暴露自己的原生家庭创伤"——这个组合就是辨识度

类比一下：可口可乐的差异化不是"甜水"（所有饮料都甜），也不是"红色"（很多品牌用红色），而是"甜水+红色+那个弧形瓶子+那种广告调性"的整体组合。

### 克莱恩法则：分享你的过程，不只是结果

回到奥斯汀·克莱恩那句话："你不需要成为天才，你只需要把你的过程分享出来。"

**为什么这句话对你特别重要？**

因为你现在处在一个很好的位置：你不是一个"已经什么都懂了"的专家在俯视观众，你是一个**正在学习、正在成长、正在从原生家庭创伤中恢复的人**。

这个"正在进行中"的状态本身就是品牌资产。

观众不只想看"答案"，还想看"一个人怎么走出来的"。如果你把自己的学习过程、犯的错误、想通的瞬间分享出来——这就是你的品牌壁垒。因为没有人能复制你的人生经历。

这和你说的"想少提自己"并不冲突。你不需要每条视频都讲自己的故事。但你的品牌底色应该是"一个从关系创伤中走出来的人，现在把她学到的东西翻译给你听"。这个底色哪怕不明说，观众也能感受到。

### 品牌过滤器：一个选题该不该做？

有了品牌三角，选题决策就变简单了。每次面对一个选题，过三个问题：

```
选题过滤器：
┌─────────────────────────────────────────────┐
│ 1. 这个选题在我的核心议题范围内吗？          │
│    → 不在 → 不做                            │
│                                              │
│ 2. 我能用我的身份定位/视角来讲吗？            │
│    → 跟我的视角无关 → 不做                   │
│                                              │
│ 3. 讲完之后，观众对我的"品牌预期"会加强       │
│    还是混乱？                                │
│    → 混乱 → 不做                            │
└─────────────────────────────────────────────┘
三个都是"是" → 做。
```

很多创作者追热点追到品牌崩塌，就是因为没有过滤器。一个热点再火，如果跟你的品牌无关，做了只会稀释你。

### 品牌建设的节奏

品牌不是一天建成的。它有阶段：

**阶段一：探索期（你现在在这里）**
- 多试不同的选题和风格
- 观察哪条视频数据好、哪条评论区活跃
- 从数据和评论中**发现**你的品牌三角
- 这个阶段允许不一致

**阶段二：确认期**
- 品牌三角基本清晰
- 开始**有意识地**围绕品牌三角创作
- 淘汰不符合品牌的选题类型
- 统一视觉风格和表达方式

**阶段三：强化期**
- 品牌已建立，观众有稳定预期
- 重点变成"在品牌框架内创新"
- 偶尔的"破格"内容反而加强品牌（因为观众已经知道你"通常是什么样"，所以"不一样"才有冲击力）

**不要在阶段一就想着阶段三的事。** 现在最重要的是多做、多看数据反馈、让品牌自然浮现。

## 小结

1. **品牌 = 观众对你的稳定预期**，不是包装，是承诺——"看我的内容你会得到什么"
2. **品牌不需要"造"，需要"发现"** — 回看你反复在讲什么、别人因为什么来找你、你愿意一直做的是什么
3. **品牌三角**（身份定位 × 核心议题 × 表达风格）把方向变成可操作的框架
4. **差异化 = 可被辨认**，不是单一特征，而是品牌三角组合的辨识度
5. **分享过程，不只是结果** — 你"正在成长中"的状态本身就是品牌壁垒
6. **品牌过滤器**：核心议题内？我的视角能讲？品牌预期加强还是混乱？三个"是"才做
