# 传媒学 - 第 9 课补强练习

## 背景

第09课检查站得分 69/100（部分理解）。概念理解了，问题在应用——回答停在结论层，没有展开"为什么"。

## 补强练习

### 第1题重答：品牌三角（每个角至少3句话）

**身份定位：** 用课程里的四种身份（专家/亲历者/翻译官/观察者）来定义你的角度。可以是组合。要具体到"能指导你下一条视频怎么拍"的程度。

**核心议题：** 用一句话概括，要求：比"心理学"窄，比"NPD"宽，能当选题过滤器用。然后解释为什么选这个范围。

**表达风格：** 覆盖三个维度——
- 语言层面（说话方式、称呼、句式）
- 视觉层面（镜头、布景、配色）
- 情绪基调（温暖型/锐评型/反差型？）

### 第2题重答：品牌过滤器（每个选题过三个问题）

三个选题：
1. 孝顺是两个词
2. 能量低是你处理不好关系的原因
3. 边界感不强，胖三斤

每个选题回答：

| 过滤问题 | 是/否 | 一句理由 |
|----------|-------|----------|
| 这个选题在我的核心议题范围内吗？ | | |
| 我能用我的身份定位/视角来讲吗？ | | |
| 讲完后观众对我的品牌预期加强还是混乱？ | | |

结论：做 / 不做 / 做但需要调整（说明调什么）
