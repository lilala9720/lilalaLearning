# 传媒学 - 第 10 课补强练习

## 背景

第10课检查站得分 70/100（部分理解）。自我诊断能力好，但两个核心问题需要补强：
1. 把"受众画像清晰"混淆为"内容-受众匹配"
2. 漏斗诊断时给出的多个原因实质是同一个方向

## 补强练习

### 第1题：内容-受众匹配的真实检验

下面三个创作者，哪一个达到了内容-受众匹配？哪一个没有？用课程里的标准判断，并说明理由。

**创作者A：** 粉丝 8 万，女性占 75%，18-30岁为主。最近10条视频平均播放 2 万。评论区以"哈哈哈""笑死"为主。她停更两周，没有人问她去哪了。

**创作者B：** 粉丝 3000，女性占 80%。最近10条视频平均播放 5000。评论区经常出现"姐姐你能不能讲一下XX""你上次说的那个方法我试了，真的有用"。她停更一周，有十几个人在最新视频下留言"终于等到你了"。

**创作者C：** 粉丝 1.2 万，观众画像分散，男女各半，年龄层跨度大。最近10条视频播放量波动剧烈（500到5万不等）。偶尔一条爆了，但下一条又回到几百播放。

### 第2题：用匹配标准审视你自己

回到你自己的账号。不要用"女性是男性3倍"来判断。用以下三个具体信号逐条检查：

| 匹配信号 | 你的情况（有/没有/不确定） | 具体证据 |
|----------|--------------------------|----------|
| 评论区有人说"你说的就是我的情况"或类似表达 | | |
| 有人翻你历史视频连看多条（看后台数据，或有人评论老视频） | | |
| 有重复出现的评论者（同一个人在多条视频下互动） | | |

根据这三个信号，重新判断：你处于匹配的哪个阶段？
- A. 还没匹配（几乎没有以上信号）
- B. 早期匹配（有零星信号，但不稳定）
- C. 已匹配（三个信号都有，且持续出现）

### 第3题：同一个数据，三个不同方向的假设

场景：你发了一条"孝顺是两个词"的视频，播放量 3 万，完播率 45%，点赞率 8%，但关注转化率只有 0.03%。
