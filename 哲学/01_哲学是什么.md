# 哲学 - 第 1 课：哲学是什么——问题、方法与分支

## 学习目标

学完本课后，你能够：
1. 清晰说出哲学与科学、宗教、常识的区别
2. 理解哲学的核心方法：概念分析与论证
3. 画出哲学的"地图"——知道主要分支各自在问什么

---

## 内容讲解

### 一、哲学不是什么

很多人对哲学的第一印象是"想一些没用的问题"或者"人生感悟"。先破除三个常见误解：

| 误解 | 实际上 |
|------|--------|
| 哲学 = 心灵鸡汤 | 哲学要求严格论证，不接受"我觉得" |
| 哲学 = 科学还没解决的问题 | 有些问题**原则上**不是科学能回答的（比如"科学方法本身为什么可靠？"） |
| 哲学 = 每个人都有自己的哲学 | 你可以有观点，但哲学要求你**为观点给出理由，并接受反驳** |

### 二、哲学的独特之处：二阶追问

科学问"世界是怎样的"，哲学问"我们凭什么这么说"。

**例子：**
- 物理学说"光速不变"→ 哲学问：**"物理定律是发现的还是发明的？"**
- 法官判"这是不正义的"→ 哲学问：**"正义的标准从哪来？"**
- 心理学说"他有意识"→ 哲学问：**"什么叫'有意识'？你怎么知道别人有意识？"**

这就是所谓**"二阶思考"**——不只是回答问题，而是追问"这个问题本身意味着什么"。

你在诊断测试里问"我有没有做这个选择的权力"——这就是一个典型的二阶追问，你天然就会做这件事。

### 三、哲学的核心方法

哲学家不做实验，他们的"实验室"是**论证（argument）**。一个哲学论证长这样：

```
前提1：所有人都会死。
前提2：苏格拉底是人。
结论：所以，苏格拉底会死。
```

哲学的工作就是：
1. **概念分析**：搞清楚词语到底是什么意思（比如"自由""正义""知识"）
2. **构造论证**：用前提推出结论
3. **寻找反例**：试图推翻别人（或自己）的论证
4. **思想实验**：构造假想场景来测试直觉（比如电车问题）

### 四、哲学的地图——六大分支

把哲学想象成一棵树：

```
                    哲学
                     │
        ┌────┬───┬───┼───┬────┬────┐
        │    │   │   │   │    │    │
      形而上学 认识论 伦理学 逻辑学 美学 政治哲学
       │      │     │     │    │    │
    "什么  "怎么  "该怎  "怎么 "什么 "怎样的
     存在？" 知道？" 么做？" 推理？" 是美？" 社会是
                                         好的？"
```

| 分支 | 核心问题 | 你已经碰过的例子 |
|------|---------|-----------------|
| **形而上学** | 什么东西存在？事物的本质是什么？ | 斧头换了还是不是原来的（忒修斯之船） |
| **认识论** | 知识是什么？我们能确定什么？ | "眼见是否为实" |
| **伦理学** | 什么是对的？我该怎么做？ | 电车问题 |
| **逻辑学** | 什么算有效的推理？ | （下节课重点） |
| **美学** | 什么是美？艺术的本质是什么？ | （后面会涉及） |
| **政治哲学** | 正义的社会是什么样的？ | "我有没有权力做选择"其实就触及了这个 |

### 五、哲学与相邻学科的关系

```
        宗教：信仰 → "信就好了"
         │
  哲学：理性论证 → "给我理由"
         │
        科学：经验检验 → "给我证据"
```

- **宗教**诉诸信仰和权威
- **哲学**诉诸理性和论证
- **科学**诉诸经验和实验

它们不是对立的，而是用不同的工具处理不同层次的问题。科学告诉你基因编辑技术上可行，哲学追问"我们应不应该编辑人类基因"。

---

## 小结

1. **哲学的核心是"二阶追问"**——不只是回答问题，而是追问问题本身的前提和概念
2. **哲学的方法是论证**——概念分析、构造推理、寻找反例、思想实验
3. **哲学有六大分支**，各自追问不同的根本问题
4. **哲学不是"想到什么说什么"**，它要求为自己的观点给出理由，并且愿意被反驳
5. **你已经有哲学直觉了**——诊断测试中你做的很多事（质疑前提、区分角度）就是哲学思维
