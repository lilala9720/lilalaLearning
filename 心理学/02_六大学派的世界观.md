# 心理学 - 第 2 课：六大学派的世界观

## 学习目标

学完这节课，你能：
- 说出心理学六大主要视角各自的核心主张
- 理解它们之间不是"谁对谁错"，而是"看的角度不同"
- 把你已经知道的心理学概念归位到对应学派

---

## 内容讲解

### 先打一个比方

假设有个人在路上突然大哭。六个不同学派的心理学家经过，他们会怎么解释？

| 学派 | 他会怎么说 |
|------|-----------|
| 精神分析 | "他的潜意识里有未处理的童年创伤被触发了" |
| 行为主义 | "他之前在这个场景下哭过并得到了安慰（强化），所以这个行为被保留了" |
| 人本主义 | "他的某个核心需求没被满足，他在经历真实的痛苦" |
| 认知 | "他对某个事件的**解读方式**让他觉得无法承受" |
| 生物学 | "他的杏仁核过度激活，可能血清素水平偏低" |
| 进化 | "哭泣是一种社会信号，是进化出来的求助机制" |

六个人说的都有道理，但他们**看的东西完全不同**。这就是学派的本质——不是答案不同，是**问题不同**。

---

### 一、精神分析（Psychoanalysis）

**创始人**：弗洛伊德（Freud），19世纪末

**核心主张**：人的行为由**潜意识**驱动。你以为你在做理性决定，其实你的童年经历、压抑的欲望、未解决的冲突在暗中操控你。

**关键概念**：
- **本我（id）**：原始欲望，"我要！现在就要！"
- **自我（ego）**：现实协调者，"等等，想个办法"
- **超我（superego）**：道德警察，"你不应该！"
- **防御机制**：当内心冲突太痛苦，自我用各种方式保护你——否认、投射、合理化、升华……

**贡献**：第一个认真对待"人不完全理性"的学派。潜意识、防御机制、童年经历的长期影响——这些概念已经渗透到日常语言里。

**局限**：很多核心概念**不可证伪**。你说一个人有攻击性是因为"潜意识里的死本能"——这怎么验证？怎么推翻？如果不能被推翻，就不是科学命题（还记得第1课的判断标准吗？）。

> 你做短视频时讲过"防御机制"吗？如果讲过，那就是精神分析学派的概念。

---

### 二、行为主义（Behaviorism）

**代表人物**：华生（Watson）、斯金纳（Skinner），20世纪初-中期

**核心主张**：别猜人心里在想什么，**只研究可观察的行为**。行为是环境塑造的——给什么刺激，就出什么反应。

**关键概念**：
- **经典条件反射**（巴甫洛夫）：铃声 + 食物重复配对 → 听到铃声就流口水。刺激之间的关联学习。
- **操作性条件反射**（斯金纳）：行为之后有奖励→行为增加；行为之后有惩罚→行为减少。
  - **正强化**：给好东西（发奖金→更努力工作）
  - **负强化**：拿走坏东西（吃药→头不痛了→以后头痛就吃药）。注意：负强化**不是惩罚**。
  - **惩罚**：给坏东西或拿走好东西（罚款、没收手机）

**贡献**：让心理学变得可量化、可实验。强化和惩罚的原理至今仍是理解行为的基础工具。

**局限**：上节课说过了——解释不了语言习得、创造性思维等"没被刺激过但能产生"的行为。把人简化成了一台反应机器。

> **常见混淆**："负强化"不是"惩罚"。负强化是让行为**增加**的（因为移除了不舒服的东西），惩罚是让行为**减少**的。很多人搞混这个。

---

### 三、人本主义（Humanistic Psychology）

**代表人物**：马斯洛（Maslow）、罗杰斯（Rogers），1950s-1960s

**核心主张**：精神分析太阴暗（整天讲创伤和潜意识），行为主义太机械（把人当老鼠）。人有**自我实现**的内在动力，心理学应该研究健康、成长和潜能。

**关键概念**：
- **马斯洛需求层次**：生理→安全→归属→尊重→自我实现。低层没满足，高层就顾不上。
- **罗杰斯的无条件积极关注**：人在被完全接纳（不加评判）时，才能真正成长。这是人本主义心理咨询的核心。
- **自我实现**：成为你能成为的最好版本。

**贡献**：把心理学的关注点从"治病"转向"成长"。罗杰斯的来访者中心疗法至今是咨询的基石之一。

**局限**：概念比较"软"，难以精确定义和测量。"自我实现"到底怎么量化？"无条件积极关注"的边界在哪？科学性不够硬。

---

### 四、认知心理学（Cognitive Psychology）

**兴起**：1950s-1970s 认知革命

**核心主张**：行为主义忽略的那个"黑箱"（大脑内部加工）才是关键。人不是被动接受刺激，而是**主动解读**信息。同样的事件，不同的人反应不同，是因为他们的**认知加工方式**不同。

**关键概念**：
- **图式（schema）**：大脑里的"模板"，你用它来快速理解新信息。比如你有一个"餐厅"图式——进门、坐下、看菜单、点菜——去任何新餐厅都能快速适应。
- **认知偏差**：系统性的思维错误（你摸底时不知道的那个）
- **信息加工模型**：输入→编码→存储→提取，类似计算机

**贡献**：打开了"黑箱"，让思维、记忆、注意、决策等内部过程变成可研究的对象。CBT（认知行为疗法）就是认知学派的直接产物。

**局限**：有时过于强调"理性加工"，对情绪、动机、身体感受的关注不够。人不总是在"思考"，有时候是在"感受"。

> 你在摸底测试里说的 CBT，就是认知学派最成功的临床应用。

---

### 五、生物学视角（Biological Perspective）

**核心主张**：心理现象的基础是**大脑和身体**。神经元怎么放电、神经递质怎么传递、大脑哪个区域负责什么功能——搞清楚这些，就能理解行为。

**关键概念**：
- **神经递质**：多巴胺（奖赏和动机）、血清素（情绪调节）、去甲肾上腺素（警觉）等化学信使
- **脑区功能定位**：杏仁核管恐惧、海马管记忆、前额叶管决策和自控
- **基因与行为**：双生子研究揭示了基因对人格、智力、心理障碍的影响

**贡献**：让心理学有了"硬件"层面的解释。抗抑郁药、抗焦虑药的研发都基于这个视角。fMRI 等技术让我们能"看见"心理活动。

**局限**：容易**还原论**——把复杂的人类体验简化成"不就是化学反应嘛"。一个人失恋后抑郁，你说"血清素低了"，这是解释还是回避？

---

### 六、进化心理学（Evolutionary Psychology）

**核心主张**：人类的心理机制是自然选择的产物。我们的大脑是在远古环境（狩猎采集时代）中进化出来的，很多看似"不合理"的行为，用进化逻辑就能解释。

**关键概念**：
- **适应性问题**：我们的祖先面临生存和繁殖的挑战（找食物、躲猎食者、选配偶），心理机制是为了解决这些问题而进化的
- **错配理论**：大脑是为石器时代设计的，但我们生活在现代社会。爱吃高糖高脂食物？因为远古时代这些资源稀缺，囤积热量有生存优势。如今超市随处有，但大脑还在用旧程序。

**贡献**：提供了"终极原因"的解释——不只问"人为什么这样做"（近因），还问"这个心理机制为什么会存在"（远因）。

**局限**：很多解释是**事后叙事**，难以验证。"男人花心是因为进化要最大化基因传播"——这和"常识"一样，什么结果都能圆回来。而且容易被滥用来为不良行为开脱（"天性如此"）。

---

### 总结：一张地图

```
          看"内心深处"                    看"可观察行为"
               │                              │
          精神分析                          行为主义
          (潜意识驱动)                     (环境塑造)
               │                              │
               └──── 都被批评太极端 ────────┘
                           │
                    认知心理学（折中）
                   "内部加工过程很重要"
                           │
            ┌──────────────┼──────────────┐
            │              │              │
        人本主义       生物学视角      进化心理学
      (关注成长)     (关注大脑)     (关注适应性)
```

今天的心理学不站任何一个学派，而是根据研究的问题**选择最合适的视角**，甚至多个视角同时使用。

---

## 小结

1. 六大视角各自回答不同的问题：精神分析看潜意识，行为主义看环境，人本看成长，认知看思维加工，生物看大脑，进化看适应性
2. 没有"最正确"的学派，只有"最适合当前问题"的视角
3. 每个学派都有贡献也有局限——能说出局限，才算真正理解了这个学派
4. 你做短视频时用到的概念（防御机制、依恋、CBT……）分属不同学派，现在可以归位了
