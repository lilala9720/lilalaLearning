# 维特根斯坦 - 第 4 课：转折——从剑桥回归到推翻自己

## 学习目标

学完本课后，你能够：
- 说出维特根斯坦"中间期"（1929-1936）发生了什么，他为什么重返哲学
- 理解《逻辑哲学论》的体系是怎么裂开的——不是被别人攻破的，而是被他自己发现了漏洞
- 具体说出至少两个让图像论崩塌的反例
- 感受从前期到后期的转变不是"换了个想法"，而是整个哲学方法的革命

## 内容讲解

### 他本来以为自己退休了

1921年，《逻辑哲学论》出版。维特根斯坦觉得哲学问题已经全部解决了，于是他：

- 把遗产全部送给兄弟姐妹
- 去奥地利乡下当了六年小学教师（1920-1926）
- 给姐姐设计了一栋房子（现在是保加利亚大使馆）
- 当了一阵园丁

一个"终结了哲学"的人去教小孩算术和拼写。这本身就很维特根斯坦。

### 他为什么回来了？

1929年，维特根斯坦重返剑桥。是什么让他觉得自己错了？

有几个触发点：

**触发点一：和拉姆齐的对话**

弗兰克·拉姆齐（Frank Ramsey）是剑桥的天才数学家，也是《逻辑哲学论》英文版的审校者。他在和维特根斯坦的多次讨论中指出了一些技术性问题。这些对话让维特根斯坦开始怀疑自己的体系是否真的那么完美。

**触发点二：布劳威尔的讲座**

1928年，维特根斯坦听了数学家布劳威尔（Brouwer）关于数学基础的讲座。布劳威尔的直觉主义数学主张"数学不是发现，而是人的构造活动"——这个想法深深触动了维特根斯坦，让他开始重新思考语言和逻辑的本质。

**触发点三：他自己在教小学时的经验**

这一点很少有人强调，但可能是最重要的。维特根斯坦教了六年小学。他每天面对的是：孩子们怎么学会使用语言？他们不是先学会逻辑规则再说话的——他们是在**做事情**的过程中学会说话的。

这个日常观察，和《逻辑哲学论》的预设（语言有一个统一的逻辑结构）产生了巨大的冲突。

### 图像论到底哪里裂了？

不是一个大洞，而是好几条裂缝同时出现：

**裂缝一：颜色问题**

《逻辑哲学论》说，基本命题之间是**逻辑独立**的——一个基本命题为真或为假，不影响其他基本命题。

但考虑这个：
- "这个点是红色的"
- "这个点是蓝色的"

这两个命题不能同时为真（一个点不能同时既是纯红又是纯蓝）。但它们都是描述事实的基本命题。按照《逻辑哲学论》的体系，基本命题之间应该互相独立，不应该有这种互斥关系。

这意味着：**现实中事物的性质之间存在逻辑关系，而《逻辑哲学论》的框架装不下这种关系。**

就像你发现一张地图上标注的两条路不可能同时通行——问题不在于路，而在于地图的画法有缺陷。

**裂缝二：语言的多样性**

图像论假设语言的本质功能是"描述事实"。但维特根斯坦开始注意到，语言的用途远远不止描述：

- "把门关上！"——命令
- "今天天气真好啊"——感叹（不一定是在传递天气信息）
- "我保证明天来"——承诺（说这句话本身就是在做某事）
- "嗯嗯""哦""啊"——维持对话
- "1, 2, 3, 开始！"——协调行动

这些都是语言，但没有一个是在"画事实的图像"。如果语言只是画图工具，那这些用法全都是"无意义的"。但它们明明非常有用，非常重要。

**裂缝三：规则的执行问题**

《逻辑哲学论》假设语言有一套确定的逻辑规则。但维特根斯坦开始追问：**规则是怎么被"遵循"的？**

比如有人教你一个数列：2, 4, 6, 8, ...让你继续。你说10。但谁规定了"10"是对的？也许这个数列的规则不是"+2"，而是"前4项+2，之后+3"，那下一个就应该是11。你说"显然是+2"——但"显然"是从哪来的？

规则本身不能决定自己的应用。你需要另一个规则来解释第一个规则怎么用，然后还需要第三个规则来解释第二个……这是一个无限后退。

这个问题后来成为他后期哲学的核心主题之一。

### 转折的本质：从"看"到"用"

总结一下，维特根斯坦的转变可以用一句话概括：

**前期：语言的意义在于它"画"了什么（图像）**
**后期：语言的意义在于它"做"了什么（使用）**

这不是修修补补，而是整个哲学方法的革命：

| | 前期 | 后期 |
|--|------|------|
| 语言是什么 | 世界的逻辑图像 | 一种社会活动 |
| 意义来自哪 | 命题与事实的映射关系 | 语言在生活中的使用方式 |
| 哲学方法 | 逻辑分析 | 观察语言的实际用法 |
| 理想 | 找到语言的统一本质 | 放弃"本质"，接受多样性 |
| 类比 | 语言像一面镜子 | 语言像一个工具箱 |

维特根斯坦后来说了一句著名的话：

> "不要想，去看！"（Don't think, look!）

意思是：不要坐在椅子上想"语言的本质是什么"，去**看**人们实际上是怎么使用语言的。答案不在理论里，在现实里。

## 小结

1. **维特根斯坦因为多重因素重返哲学**：和拉姆齐的讨论、布劳威尔的讲座、以及最重要的——六年小学教学经验让他看到语言的实际运作和《逻辑哲学论》的理论不符
2. **颜色问题**暴露了图像论的技术缺陷：基本命题之间的逻辑独立性假设不成立
3. **语言的多样性**推翻了"语言本质是描述"的假设：命令、感叹、承诺、寒暄都是语言，但都不是在"画图"
4. **规则的无限后退**问题动摇了"语言有确定逻辑规则"的根基：规则不能自己解释自己
5. **核心转变是从"看"到"用"**：语言的意义不在于它像什么，而在于它被用来做什么
