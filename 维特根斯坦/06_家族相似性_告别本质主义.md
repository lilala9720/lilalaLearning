# 维特根斯坦 - 第 6 课：家族相似性——告别本质主义

## 学习目标

学完本课后，你能够：
- 解释"家族相似性"（Familienähnlichkeit）这个概念，以及它为什么是对"本质主义"的根本挑战
- 理解为什么维特根斯坦认为"所有X都有一个共同本质"这种想法是哲学困惑的主要来源
- 用家族相似性的视角分析至少一个日常概念
- 感受这个思想对心理学、社会科学甚至日常思维的冲击

## 内容讲解

### 一个两千年的习惯

从柏拉图开始，西方哲学就有一个根深蒂固的习惯：**给事物找本质。**

- "美是什么？"——一定有一个所有美的事物共享的"美的本质"
- "正义是什么？"——一定有一个所有正义行为共享的"正义的本质"
- "知识是什么？"——一定有一个区分知识和非知识的"本质特征"

这种思维方式叫**本质主义（Essentialism）**：每个概念背后都有一个固定的、统一的"核心定义"，找到它就等于理解了这个概念。

维特根斯坦说：**这个习惯本身就是错的。**

### "游戏"的本质是什么？

上一课我们已经碰过这个问题。维特根斯坦问：所有被叫做"游戏"的东西，有什么共同特征？

他邀请你真的去**看**（不要想，去看！）：

| 游戏 | 有规则？ | 有输赢？ | 有对手？ | 需要技巧？ | 为了娱乐？ |
|------|---------|---------|---------|-----------|-----------|
| 国际象棋 | ✓ | ✓ | ✓ | ✓ | 不一定 |
| 足球 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 单人纸牌 | ✓ | 有点 | ✗ | 有点 | ✓ |
| 小孩扔球玩 | 不太有 | ✗ | ✗ | 不太需要 | ✓ |
| 过家家 | 模糊 | ✗ | ✗ | ✗ | ✓ |
| 俄罗斯轮盘赌 | ✓ | ✓ | 有点 | ✗ | ✗ |

你找不到**任何一个特征**是所有游戏都有的。但你也不会因此说"过家家不是游戏"或"俄罗斯轮盘赌不算游戏"。

那"游戏"这个词到底靠什么把这些东西聚在一起？

### 家族相似性：一张交叉重叠的网

维特根斯坦的回答：

> "我想不出比'家族相似性'更好的词来描述这些相似性之间的关系了。因为一个家庭的成员之间的各种相似性——体格、面部特征、眼睛的颜色、步态、性情等等——就是以这种方式交叉重叠的。"（§67）

想象一个大家庭的合照：
- 爷爷和爸爸：同样的鼻子
- 爸爸和你：同样的眼睛
- 你和妹妹：同样的嘴巴
- 妹妹和爷爷：同样的步态

**没有一个特征是所有人都有的**，但每两个人之间都有**某些**相似之处。这些相似性交叉重叠，编成了一张网——这张网就是"家族"。

"游戏"也是这样：

```
国际象棋 ←有规则有输赢→ 足球 ←需要身体运动→ 小孩扔球 ←为了好玩→ 过家家 ←假装/角色扮演→ 电子RPG ←有规则有输赢→ 国际象棋
```

每两个相邻的游戏之间共享某些特征，但链条两端的游戏（国际象棋和过家家）可能几乎没有共同点。它们被叫做"游戏"，不是因为共享本质，而是因为它们通过**家族相似性**连接在一起。

### 这对哲学的冲击

这个想法看似简单，实际上是在拆除哲学2000年来的基本操作。

**柏拉图式的问题**："美是什么？""正义是什么？""知识是什么？"——这些问题预设了"美""正义""知识"各自有一个统一的本质。

**维特根斯坦的回应**：这些问题可能从一开始就问错了。"美"不是一个有本质的东西，它是一组通过家族相似性连接的使用方式。你觉得困惑，不是因为"美的本质"太深奥难以发现，而是因为**根本不存在这样一个本质**——你在寻找一个不存在的东西，当然会困惑。

这就是维特根斯坦说的"哲学是治疗"——他不是给你一个关于"美"的答案，他是帮你**看清这个问题本身的结构有问题**。

### 对日常思维的影响

家族相似性不只是一个哲学概念，它对日常思维有直接冲击：

**例子1：什么是"心理健康"？**

你能给"心理健康"下一个精确定义吗？快乐？适应社会？自我实现？没有症状？这些标准之间有重叠但不完全一致。一个隐居的艺术家不太"适应社会"但可能非常"自我实现"。"心理健康"不是一个有固定本质的概念——它是一组家族相似的状态。

**例子2：什么算"艺术"？**

杜尚把一个小便池放进美术馆，它是艺术吗？一幅精美的油画是艺术，街头涂鸦呢？AI生成的画呢？你找不到"艺术的本质"，但你也不需要——"艺术"这个概念就是一张家族相似性编成的网，它的边界是模糊的，而且**本来就该是模糊的**。

**例子3：什么是"游戏"？——再来一次**

电子游戏刚出现时，很多人说"这不是真正的游戏"。为什么？因为他们脑子里有一个"游戏的本质"（比如必须有身体运动、必须面对面）。但按照家族相似性的思路，电子游戏和传统游戏之间有足够多的相似线索（规则、挑战、娱乐、技巧），所以它完全有资格加入"游戏"这个家族。

### 边界模糊不是缺陷，是特征

本质主义者会说：概念的边界应该清晰，模糊的概念是有缺陷的。

维特根斯坦反驳：

> "我说'地面的那边'。这难道就是完全无用的吗，仅仅因为我没有画出精确的边界？"（§71）

你说"站在那边"，对方完全能听懂——即使"那边"没有精确的毫米级边界。概念的模糊不是bug，是feature。**人类语言就是靠这种弹性来应对无限复杂的现实的。**

如果每个词都必须有精确定义才能使用，人类根本无法说话。

## 小结

1. **本质主义**认为每个概念背后都有一个固定的、统一的"本质"——维特根斯坦认为这是2000年来哲学困惑的主要来源
2. **家族相似性**：一组事物被同一个词指称，不是因为它们共享某个共同本质，而是因为它们之间存在交叉重叠的相似性网络
3. **"不要想，去看！"**：不要坐在椅子上推演"游戏的本质是什么"，去看人们实际上把哪些东西叫做游戏
4. **概念的边界天然是模糊的**，这不是缺陷而是特征——正是这种弹性让语言能够应对无限复杂的现实
5. **哲学困惑的治疗**：当你问"X的本质是什么"却怎么也找不到答案时，也许该问的不是"答案是什么"，而是"这个问题本身是否合理"
