# 你的门店/团队，是在"等客户买"还是"让客户想买"？

---

## 五个问题，测出你团队的真实战力

1. **客户进店逛了一圈没买就走了，你的销售能说出为什么吗？**
   不知道客户为什么走，就永远不知道怎么让他留下。

2. **客户说"我再看看"，你的销售是放人走还是有下一步动作？**
   90%的销售在这一句话之后就失去了这个客户。

3. **同样的产品、同样的价格，为什么有人月销百万有人月销十万？**
   差距不在产品，在于客户走进来之后的那 5 分钟。

4. **你的销售是在介绍参数，还是在描述客户拥有之后的生活？**
   客户买的不是产品，是产品带来的那个"更好的自己"。

5. **复购和转介绍占你业绩的多少？**
   如果低于 30%，说明你一直在花钱买新客，而不是经营老客。

## 培训框架：To C 成交四步法

```
  ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
  │ 破冰连接  │ →  │ 需求激发  │ →  │ 体验促单  │ →  │ 锁客复购  │
  └──────────┘    └──────────┘    └──────────┘    └──────────┘
   3秒建立好感     不是问需要什么     让客户自己说服     一次成交只是
   打破防备心理     而是帮他发现       自己"我要这个"     长期关系的开始
```

> 每一步都有对应的话术模板、场景演练和实操工具。

---

## 核心理念

> **"顶级销售从不推销——他们只是让客户看见了自己内心本来就想要的东西。"**

培训的目标，不是教你的人"更会说"，
而是让每个客户走进来都觉得"这个人懂我"。

---

## 培训能带来什么？

| 维度 | 变化 |
|------|------|
| 转化率 | 提升，进店客户不再白白流失 |
| 客单价 | 拉高，从卖一件到连带成交 |
| 复购率 | 增长，客户主动回来找你买 |
| 人效差距 | 缩小，新人也能快速上手出单 |
