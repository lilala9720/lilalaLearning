# 你的销售团队，离"可复制的赢单系统"还有多远？

---

## 五个问题，测出你团队的真实战力

1. **客户说"我们内部再讨论一下"，你的销售下一步做什么？**
   大多数人在这一步丢单，却以为自己还"在跟进"。

2. **团队里最能签单的人走了，业绩会塌多少？**
   如果答案是"会塌很多"，说明你卖的是个人能力，不是系统。

3. **你的销售能在 30 秒内说清楚，客户为什么要选你而不是竞品吗？**
   说不清价值，就只能卷价格。

4. **从第一次接触到签约，你能说出平均经过几个关键节点吗？**
   管不了节点，就管不了预测，管不了预测，就管不了增长。

5. **客户的"决策链"上有几个人？你的销售见了几个？**
   To B 的单子不是一个人拍板的，只搞定一个人 = 把命运交给别人。

## 培训框架：B2B 赢单五步法

```
  ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
  │ 精准触达  │ →  │ 需求深挖  │ →  │ 价值锚定  │ →  │ 决策推进  │ →  │ 赢单闭环  │
  └──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
   找对人           问对事           说对话           推对人           收对钱
   不浪费子弹       不停在表面       不陷入比价       不卡在流程       不功亏一篑
```

> 每一步都有对应的工具、话术模板和实战演练方案。

---

## 核心理念

> **"B2B 销售的本质不是说服采购，是帮客户内部达成共识。"**

你的销售不只是在卖产品——他们是客户内部的"编外项目经理"。
培训的目标，是让每个销售都能扮演好这个角色。

---

## 培训能带来什么？

| 维度 | 变化 |
|------|------|
| 成交周期 | 缩短，因为不再在错误的人身上浪费时间 |
| 赢单率 | 提升，因为每一步都有明确动作 |
| 人才依赖 | 降低，因为方法可复制、可考核 |
| 销售预测 | 可靠，因为节点清晰、数据可追踪 |
